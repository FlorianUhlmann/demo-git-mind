# Ideas

This folder contains concepts that are being considered for integration into the main document but are not yet fully developed or placed.

**Each idea lives in its own file** so it can be discussed, refined, and eventually merged into the right section via PR.

## How to add an idea

1. Fork this repository
2. Create a new file in this folder: `ideas/your-concept-name.md`
3. Use the template below
4. Open a Pull Request — the owner will review whether and where it integrates

## Idea file template

```markdown
# [Idea Title]

> **Proposed placement:** Section X or new section
> **Status:** Draft / Ready for review
> **Author:** your name or handle

---

[Your idea here. Include: what it is, why it matters, how it connects to the existing document, and any open questions.]
```
